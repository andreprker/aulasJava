Sintaxe Básica de Classe e Criação de Objeto em Java

Compreenda a estrutura fundamental para definir classes e como instanciar objetos a partir delas na linguagem de programação Java.

Definição de Classe em Java: O Molde

Em Java, uma classe é definida usando a palavra-chave `class`, seguida pelo nome da classe. Atributos (variáveis) são declarados com um tipo de dado e um modificador de acesso (como `private` ou `public`). Métodos (funções) também têm um tipo de retorno, nome e modificadores de acesso. O construtor é um método especial com o mesmo nome da classe, usado para inicializar objetos. 

public class Carro {
    // Atributos (características)
    private String marca;
    private String modelo;
    private String cor;
    private int velocidadeAtual;

    // Construtor
    public Carro(String marca, String modelo, String cor) {
        this.marca = marca;
        this.modelo = modelo;
        this.cor = cor;
        this.velocidadeAtual = 0;
    }

    // Métodos (comportamentos)
    public void acelerar(int incremento) {
        this.velocidadeAtual += incremento;
        System.out.println("O " + this.marca + " " + this.modelo + " acelerou para " + this.velocidadeAtual + " km/h.");
    }

    public void travar() {
        this.velocidadeAtual = 0;
        System.out.println("O " + this.marca + " " + this.modelo + " travou.");
    }

    // Método para obter a velocidade atual (exemplo de getter)
    public int getVelocidadeAtual() {
        return this.velocidadeAtual;
    }
}

Explicação do Código:

  • `public class Carro { ... }` define a classe `Carro` como pública.
  • `private String marca;` declara um atributo `marca` do tipo `String` com acesso privado.
  • `public Carro(...)` é o construtor da classe, usado para inicializar os atributos do objeto.
  • `this` refere-se à instância atual do objeto.
  • `acelerar()` e `travar()` são métodos que definem o comportamento dos objetos `Carro`.
  • `getVelocidadeAtual()` é um método "getter" para aceder ao valor do atributo `velocidadeAtual`.

Criação de Objetos em Java: A Instância

Para criar um objeto (ou instanciar uma classe) em Java, você usa a palavra-chave `new` seguida pelo nome da classe e os argumentos para o construtor. Isso aloca memória para o novo objeto e executa o construtor. 

// Exemplo de uso da classe Carro em um método main
public class MinhaAplicacao {
    public static void main(String[] args) {
        // Criando objetos (instâncias da classe Carro)
        Carro meuCarro = new Carro("Toyota", "Corolla", "Prata");
        Carro carroAmigo = new Carro("Honda", "Civic", "Preto");

        // Acedendo a atributos do objeto (via métodos getters se privados)
        System.out.println("Meu carro é um " + meuCarro.marca + " " + meuCarro.modelo + " de cor " + meuCarro.cor + ".");
        System.out.println("O carro do meu amigo é um " + carroAmigo.marca + " " + carroAmigo.modelo + ".");

        // Chamando métodos do objeto
        meuCarro.acelerar(60);
        carroAmigo.acelerar(80);
        meuCarro.travar();

        // Acedendo a velocidade atual usando o método getter
        System.out.println("Velocidade atual do meu carro: " + meuCarro.getVelocidadeAtual() + " km/h.");
    }
}

Explicação do Código:

  • `Carro meuCarro = new Carro(...);` cria uma nova instância da classe `Carro` e atribui-a à variável `meuCarro`.
  • `new Carro(...)` invoca o construtor da classe `Carro`.
  • `meuCarro.acelerar(60);` chama o método `acelerar` no objeto `meuCarro`.
  • Acesso direto a atributos como `meuCarro.marca` é possível se o atributo for `public`. Se for `private`, é necessário usar métodos "getters" (como `getVelocidadeAtual()`).

Em Java, a classe é o modelo e o objeto é a sua concretização. A sintaxe é clara e segue os princípios da POO para criar software modular e escalável.
