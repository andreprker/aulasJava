Palavra-chave super em Java: Acessando Membros da Superclasse

Descubra a palavra-chave super em Java, essencial para acessar e interagir com os membros da superclasse dentro de uma subclasse.

O que é a Palavra-chave super?

Em Java, a palavra-chave super é uma referência que aponta para a **instância da superclasse imediata** da classe atual. Ela é usada em subclasses para distinguir entre membros (atributos e métodos) da superclasse e da subclasse, especialmente quando eles têm o mesmo nome.

A palavra-chave super é vital para a implementação correta da herança, permitindo que a subclasse construa e utilize a funcionalidade fornecida pela sua classe pai.

Principais Usos da Palavra-chave super

1. Para chamar o Construtor da Superclasse: super()

Quando uma subclasse é instanciada, o construtor da superclasse também deve ser chamado para inicializar as partes herdadas do objeto. A palavra-chave super() é usada para invocar o construtor da superclasse imediata. **Deve ser a primeira instrução** dentro do construtor da subclasse. 


// Superclasse
class Animal {
    String nome;

    public Animal(String nome) {
        this.nome = nome;
        System.out.println("Construtor da Superclasse Animal chamado: " + nome);
    }
}

// Subclasse
class Cachorro extends Animal {
    String raca;

    public Cachorro(String nome, String raca) {
        super(nome); // Chama o construtor da superclasse Animal
        this.raca = raca;
        System.out.println("Construtor da Subclasse Cachorro chamado: " + raca);
    }
}

public class Main {
    public static void main(String[] args) {
        Cachorro meuCachorro = new Cachorro("Rex", "Labrador");
        // Saída:
        // Construtor da Superclasse Animal chamado: Rex
        // Construtor da Subclasse Cachorro chamado: Labrador
    }
}

2. Para acessar Atributos da Superclasse: super.atributo

Se uma subclasse declara um atributo com o mesmo nome de um atributo na superclasse (ocultação de atributos), você pode usar super. para referenciar explicitamente o atributo da superclasse. 


class Veiculo {
    String tipo = "Veículo Genérico"; // Atributo da superclasse

    public void exibirTipoVeiculo() {
        System.out.println("Tipo na Superclasse: " + tipo);
    }
}

class Carro extends Veiculo {
    String tipo = "Carro de Passeio"; // Atributo da subclasse com o mesmo nome

    public void exibirTipos() {
        System.out.println("Tipo na Subclasse: " + tipo);          // Refere-se ao atributo da subclasse
        System.out.println("Tipo na Superclasse (via super): " + super.tipo); // Refere-se ao atributo da superclasse
    }
}

public class Main {
    public static void main(String[] args) {
        Carro meuCarro = new Carro();
        meuCarro.exibirTipos();
        meuCarro.exibirTipoVeiculo(); // Chama método da superclasse
        // Saída:
        // Tipo na Subclasse: Carro de Passeio
        // Tipo na Superclasse (via super): Veículo Genérico
        // Tipo na Superclasse: Veículo Genérico
    }
}

3. Para chamar Métodos da Superclasse: super.metodo()

Quando uma subclasse sobrescreve um método da superclasse, mas você ainda deseja chamar a implementação original da superclasse antes ou depois de adicionar a lógica específica da subclasse, use super.metodo(). 


class Forma {
    public void desenhar() {
        System.out.println("Desenhando uma forma genérica.");
    }
}

class Circulo extends Forma {
    @Override // Indica que este método sobrescreve um método da superclasse
    public void desenhar() {
        super.desenhar(); // Chama o método desenhar() da superclasse
        System.out.println("Adicionando detalhes de um círculo.");
    }
}

public class Main {
    public static void main(String[] args) {
        Circulo meuCirculo = new Circulo();
        meuCirculo.desenhar();
        // Saída:
        // Desenhando uma forma genérica.
        // Adicionando detalhes de um círculo.
    }
}
```

A palavra-chave super é uma ferramenta poderosa para gerenciar a herança em Java, garantindo a colaboração eficaz entre superclasses e subclasses!
