Declaração e Chamada de Atributos/Métodos em Java

Em Java, a programação orientada a objetos (POO) é construída em torno do conceito de classes e objetos. Cada objeto possui um estado (definido por seus atributos) e comportamentos (definidos por seus métodos). Para interagir com esses objetos, precisamos saber como declarar seus atributos e métodos, e como chamá-los.

---

1. Declaração de Atributos (Variáveis de Instância)

Atributos são as características ou propriedades de um objeto. Eles são declarados dentro da classe, mas fora de qualquer método.

Sintaxe Geral:

[modificador_acesso] [tipo_dado] nomeAtributo;
  • [modificador_acesso]: Define a visibilidade (ex: public, private, protected). Para atributos, private é a prática recomendada (encapsulamento).
  • [tipo_dado]: O tipo de dado que o atributo armazenará (ex: int, String, boolean, double).
  • nomeAtributo: O nome do atributo (seguindo a convenção camelCase).

Exemplo de Declaração de Atributos

public class Pessoa {
    // Atributos (variáveis de instância)
    private String nome;        // Atributo para armazenar o nome da pessoa
    private int idade;          // Atributo para armazenar a idade da pessoa
    public double altura;       // Atributo público (menos comum para atributos)
}
---

2. Chamada (Acesso e Modificação) de Atributos

Após um objeto ser instanciado (criado), seus atributos podem ser acessados ou modificados. Se os atributos são public, eles podem ser acessados diretamente. Se são private (o que é mais comum e recomendado), eles devem ser acessados e modificados através de métodos públicos (getters e setters).

Acesso Direto (para atributos públicos):

nomeObjeto.nomeAtributo = valor; (modificação)

tipoVariavel = nomeObjeto.nomeAtributo; (acesso)

Acesso via Métodos (Getters e Setters - para atributos privados):

Esta é a abordagem preferida, pois permite controlar como os dados são acessados e modificados.

public class Pessoa {
    private String nome;
    private int idade;

    // Getter para o nome
    public String getNome() {
        return nome;
    }

    // Setter para o nome
    public void setNome(String novoNome) {
        this.nome = novoNome;
    }

    // Getter para a idade
    public int getIdade() {
        return idade;
    }

    // Setter para a idade
    public void setIdade(int novaIdade) {
        if (novaIdade > 0) { // Exemplo de validação no setter
            this.idade = novaIdade;
        } else {
            System.out.println("Idade inválida!");
        }
    }
}

public class Main {
    public static void main(String[] args) {
        Pessoa pessoa1 = new Pessoa(); // Cria um objeto Pessoa

        // Modificando atributos via setters
        pessoa1.setNome("Ana");
        pessoa1.setIdade(25);

        // Acessando atributos via getters
        System.out.println("Nome: " + pessoa1.getNome());
        System.out.println("Idade: " + pessoa1.getIdade());

        // Tentando setar uma idade inválida
        pessoa1.setIdade(-5);
    }
}

3. Declaração de Métodos

Métodos são as ações que um objeto pode realizar. Eles são declarados dentro de uma classe e contêm o código que será executado quando o método for chamado.

Sintaxe Geral:

[modificador_acesso] [tipo_retorno] nomeMetodo([parametros]) {
    // Corpo do método
    // Instruções a serem executadas
    [return valor;]
}
  • [modificador_acesso]: Define a visibilidade (ex: public, private).
  • [tipo_retorno]: O tipo de dado que o método retorna (ex: int, String, void se não retornar nada).
  • nomeMetodo: O nome do método (seguindo a convenção camelCase).
  • [parametros]: Uma lista de variáveis que o método recebe como entrada, entre parênteses (vazio se não houver parâmetros).
  • [return valor;]: Usado para retornar um valor do tipo especificado pelo tipo_retorno.

Exemplo de Declaração de Métodos

public class Calculadora {
    // Método que não retorna valor (void) e não recebe parâmetros
    public void saudacao() {
        System.out.println("Olá! Bem-vindo à calculadora.");
    }

    // Método que retorna um int e recebe dois parâmetros int
    public int somar(int a, int b) {
        return a + b;
    }

    // Método que modifica um atributo do próprio objeto
    private int totalOperacoes = 0; // Atributo para controlar o número de operações

    public void registrarOperacao() {
        this.totalOperacoes++; // 'this' refere-se ao objeto atual
        System.out.println("Operação registrada. Total: " + this.totalOperacoes);
    }
}
---

4. Chamada de Métodos

Para executar as ações definidas por um método, você precisa chamá-lo em um objeto. Você usa o operador ponto (.) para acessar o método a partir de uma instância do objeto.

Sintaxe Geral:

nomeObjeto.nomeMetodo([argumentos]);
  • nomeObjeto: A instância do objeto na qual você quer que o método seja executado.
  • nomeMetodo: O nome do método a ser invocado.
  • [argumentos]: Os valores que você passa para os parâmetros do método, se houver.

Exemplo de Chamada de Métodos

public class Main {
    public static void main(String[] args) {
        Calculadora minhaCalc = new Calculadora(); // Cria um objeto Calculadora

        // Chamando um método sem parâmetros e sem retorno
        minhaCalc.saudacao();

        // Chamando um método com parâmetros e com retorno
        int resultadoSoma = minhaCalc.somar(10, 5);
        System.out.println("A soma é: " + resultadoSoma);

        // Chamando um método que modifica o estado do objeto
        minhaCalc.registrarOperacao();
        minhaCalc.registrarOperacao();
    }
}

Exemplo Completo: Classe 'Carro'

Vamos juntar os conceitos de declaração e chamada de atributos e métodos em uma única classe.

Classe 'Carro.java'

public class Carro {
    // 1. Declaração de Atributos (privados para encapsulamento)
    private String marca;
    private String modelo;
    private int ano;
    private double velocidadeAtual; // km/h

    // Construtor para inicializar o estado do objeto
    public Carro(String marca, String modelo, int ano) {
        this.marca = marca;
        this.modelo = modelo;
        this.ano = ano;
        this.velocidadeAtual = 0.0; // Velocidade inicial é zero
    }

    // 2. Métodos (Setters e Getters) para acesso e modificação dos atributos
    public String getMarca() {
        return marca;
    }

    public String getModelo() {
        return modelo;
    }

    public int getAno() {
        return ano;
    }

    public double getVelocidadeAtual() {
        return velocidadeAtual;
    }

    // Métodos de Comportamento
    public void acelerar(double incremento) {
        if (incremento > 0) {
            this.velocidadeAtual += incremento;
            System.out.println(modelo + " acelerou para " + velocidadeAtual + " km/h.");
        } else {
            System.out.println("Incremento de velocidade inválido.");
        }
    }

    public void frear(double decremento) {
        if (decremento > 0 && this.velocidadeAtual - decremento >= 0) {
            this.velocidadeAtual -= decremento;
            System.out.println(modelo + " freou para " + velocidadeAtual + " km/h.");
        } else if (decremento > 0 && this.velocidadeAtual - decremento < 0) {
            this.velocidadeAtual = 0;
            System.out.println(modelo + " freou e parou.");
        } else {
            System.out.println("Decremento de velocidade inválido.");
        }
    }

    public void exibirInfo() {
        System.out.println("\n--- Informações do Carro ---");
        System.out.println("Marca: " + marca);
        System.out.println("Modelo: " + modelo);
        System.out.println("Ano: " + ano);
        System.out.println("Velocidade Atual: " + velocidadeAtual + " km/h");
    }
}

Classe Principal para Teste ('Main.java')

public class Main {
    public static void main(String[] args) {
        // Criando instâncias (objetos) da classe Carro
        Carro meuCarro = new Carro("Toyota", "Corolla", 2020);
        Carro seuCarro = new Carro("Honda", "Civic", 2022);

        // Chamando métodos para exibir o estado inicial
        meuCarro.exibirInfo();
        seuCarro.exibirInfo();

        // Chamando métodos de comportamento para modificar o estado
        System.out.println("\n--- Ações com meuCarro ---");
        meuCarro.acelerar(50.0);
        meuCarro.acelerar(30.0);
        meuCarro.frear(20.0);
        meuCarro.exibirInfo();

        System.out.println("\n--- Ações com seuCarro ---");
        seuCarro.acelerar(100.0);
        seuCarro.frear(110.0); // Freia completamente
        seuCarro.exibirInfo();

        // Acessando atributos via getters
        System.out.println("\nModelo do meu carro: " + meuCarro.getModelo());
        System.out.println("Velocidade atual do seu carro: " + seuCarro.getVelocidadeAtual() + " km/h");
    }
}
---

Conclusão

A declaração e a chamada de atributos e métodos são a espinha dorsal da programação orientada a objetos em Java. Através dos atributos, definimos o estado de um objeto, e pelos métodos, definimos suas ações e como ele interage com seu próprio estado e com outros objetos. Dominar esses conceitos é fundamental para construir aplicações Java eficientes e bem estruturadas.
