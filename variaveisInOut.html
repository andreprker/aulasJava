Variáveis em Java e Como Exibi-las

Em Java, variáveis são contêineres para armazenar valores de dados. Cada variável deve ser declarada com um tipo de dado específico que indica o tipo e o tamanho dos valores que ela pode armazenar.

Tipos de Variáveis Comuns em Java

  • int: Armazena números inteiros (sem decimais), como 10, -200, 15000.
  • double: Armazena números de ponto flutuante (com decimais), como 3.14, 0.001, -25.75.
  • boolean: Armazena valores verdadeiro ou falso (true ou false).
  • char: Armazena um único caractere, como 'A', 'b', '9'.
  • String: Armazena uma sequência de caracteres (texto), como "Olá Mundo", "Java Programming". (Note que String é uma classe, não um tipo primitivo).

Declarando e Inicializando Variáveis

Para usar uma variável, você precisa declará-la (especificar seu tipo e nome) e, opcionalmente, inicializá-la (atribuir um valor a ela). 


// Declaração de variável
int idade;

// Inicialização de variável
idade = 30;

// Declaração e inicialização em uma única linha
String nome = "Maria";
double preco = 99.99;
boolean isAtivo = true;
char letra = 'C';

Exibindo Variáveis na Tela (Console)

Em Java, você usa o método System.out.println() para imprimir valores no console. Este método adiciona uma nova linha após a impressão. Você também pode usar System.out.print() se não quiser uma nova linha.
Entre os parenteses do código de exibição caso você queira exibir os valores das variáveis, você deixara apenas o nome declarado dela dentro deles.
Você também consegue exibir mensagens de texto desde que estas estejam entre aspas duplas.
Para mostrar uma frase e o valor da variável no mesmo método, basta separá-los por um sinal de +, esse sinal fará a concatenação entre texto e variáveis. 


// Declaração de variável
int idade;

// Inicialização de variável
idade = 30;

// Para exibir na tela
System.out.println("A idade é: " + idade);

O valor esperado na exibição será: 


A idade é: 30

Recebendo Entrada do Usuário

Para permitir que o usuário insira dados no console, você pode usar a classe Scanner do pacote java.util. Você precisa importá-la e criar um objeto Scanner que lerá a entrada do sistema (System.in). 


import java.util.Scanner; // Importa a classe Scanner

public class ExemploEntradaUsuario {
    public static void main(String[] args) {
        // Cria um objeto Scanner para ler a entrada do usuário
        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite seu nome: ");
        String nomeUsuario = scanner.nextLine(); // Lê uma linha de texto (nome)

        System.out.print("Digite sua idade: ");
        int idadeUsuario = scanner.nextInt(); // Lê um número inteiro (idade)

        System.out.print("Digite sua altura (ex: 1,75): ");
        double alturaUsuario = scanner.nextDouble(); // Lê um número double (altura)

        // IMPORTANTE: Consumir a quebra de linha restante após nextInt()/nextDouble()
        // Isso evita problemas com nextLine() subsequente
        scanner.nextLine();

        System.out.print("Você está aprendendo Java? (true/false): ");
        boolean aprendendoJava = scanner.nextBoolean(); // Lê um valor booleano

        // Fecha o objeto Scanner para liberar recursos
        scanner.close();

        // Exibe os dados inseridos pelo usuário
        System.out.println("\n--- Dados do Usuário ---");
        System.out.println("Nome: " + nomeUsuario);
        System.out.println("Idade: " + idadeUsuario + " anos");
        System.out.println("Altura: " + alturaUsuario + " metros");
        System.out.println("Aprendendo Java: " + aprendendoJava);
    }
}

Note o uso de scanner.nextLine(); após scanner.nextInt(); ou scanner.nextDouble();. Isso é crucial para "consumir" a quebra de linha que fica no buffer de entrada após a leitura de números, prevenindo que nextLine() leia uma string vazia imediatamente depois.

Exemplo Completo Integrado:


import java.util.Scanner; // Importa a classe Scanner

public class ExemploVariaveisComEntrada {
    public static void main(String[] args) {
        // --- Exemplo de Declaração e Exibição de Variáveis ---
        int idadeFixa = 25;
        double alturaFixa = 1.75;
        String nomeCompletoFixo = "João Silva";
        boolean estudanteFixo = true;
        char inicialNomeFixo = 'J';

        System.out.println("--- Dados Fixos ---");
        System.out.println("Nome: " + nomeCompletoFixo);
        System.out.println("Idade: " + idadeFixa + " anos");
        System.out.println("Altura: " + alturaFixa + " metros");
        System.out.println("É estudante? " + estudanteFixo);
        System.out.println("Inicial do nome: " + inicialNomeFixo);
        System.out.println("\n---------------------\n");

        // --- Exemplo de Recebimento de Entrada do Usuário ---
        Scanner scanner = new Scanner(System.in); // Cria um objeto Scanner

        System.out.print("Por favor, digite seu NOME: ");
        String nomeUsuario = scanner.nextLine(); // Lê o nome

        System.out.print("Agora, digite sua IDADE: ");
        int idadeUsuario = scanner.nextInt(); // Lê a idade

        // Consome a quebra de linha restante após nextInt()
        scanner.nextLine();

        System.out.print("Por fim, digite sua PROFISSÃO: ");
        String profissaoUsuario = scanner.nextLine(); // Lê a profissão

        // Fecha o objeto Scanner
        scanner.close();

        // Exibe os dados inseridos pelo usuário
        System.out.println("\n--- Dados Inseridos Pelo Usuário ---");
        System.out.println("Seu nome é: " + nomeUsuario);
        System.out.println("Sua idade é: " + idadeUsuario + " anos");
        System.out.println("Sua profissão é: " + profissaoUsuario);
    }
}
