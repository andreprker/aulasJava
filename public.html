public: Acessível de qualquer lugar em Java

Descubra o modificador de acesso public em Java e como ele garante a visibilidade universal para suas classes, atributos e métodos.

O que significa public?

Em Java, public é um modificador de acesso que indica que uma classe, atributo (variável de instância) ou método é **acessível de qualquer lugar**. Isso significa que ele pode ser acessado:

  • Dentro da própria classe.
  • Por outras classes no mesmo pacote.
  • Por classes em pacotes diferentes.
  • Por subclasses em qualquer pacote.

É o modificador de acesso menos restritivo e é usado para expor funcionalidades que devem ser globalmente disponíveis e reutilizáveis.

Por que usar public?

  • Exposição de APIs: Métodos e classes que fazem parte da interface pública de uma biblioteca ou framework devem ser public para que outros desenvolvedores possam utilizá-los.
  • Pontos de Entrada: O método main, que é o ponto de entrada de um programa Java, deve ser public para ser chamado pela Java Virtual Machine (JVM).
  • Acessibilidade Universal: Quando você deseja que uma funcionalidade ou dado esteja disponível para qualquer parte do seu sistema ou para sistemas externos que interagem com o seu código.
  • Reutilização: Componentes projetados para serem reutilizados em diferentes partes de uma aplicação ou em diferentes projetos frequentemente têm partes public.

Exemplos de Uso da Palavra-chave public

Classes public

Uma classe public pode ser acessada de qualquer outro pacote ou classe. É a forma mais comum de definir classes que servem como componentes principais de um sistema. 


// Arquivo: meu_pacote/MinhaClassePublica.java
package meu_pacote;

public class MinhaClassePublica { // Acessível de qualquer lugar
    public String mensagem;

    public MinhaClassePublica(String msg) {
        this.mensagem = msg;
    }

    public void exibirMensagem() {
        System.out.println("Mensagem da Classe Pública: " + this.mensagem);
    }
}

// Arquivo: outro_pacote/UsoDaClassePublica.java
package outro_pacote;

import meu_pacote.MinhaClassePublica; // Importação necessária

public class UsoDaClassePublica {
    public static void main(String[] args) {
        MinhaClassePublica obj = new MinhaClassePublica("Olá do mundo público!");
        obj.exibirMensagem(); // Saída: Mensagem da Classe Pública: Olá do mundo público!
        System.out.println("Acessando variável pública: " + obj.mensagem); // Saída: Acessando variável pública: Olá do mundo público!
    }
}

Atributos public

Um atributo public pode ser lido e modificado diretamente por qualquer código que tenha acesso ao objeto. Embora seja acessível, para o encapsulamento, geralmente se prefere usar métodos "getter" e "setter" para acessar atributos, mesmo que sejam públicos. 


class Produto {
    public String nome;    // Atributo público
    public double preco;   // Atributo público

    public Produto(String nome, double preco) {
        this.nome = nome;
        this.preco = preco;
    }
}

public class Main {
    public static void main(String[] args) {
        Produto livro = new Produto("Java para Iniciantes", 49.99);
        System.out.println("Nome do Produto: " + livro.nome);   // Acesso direto ao atributo público
        System.out.println("Preço: " + livro.preco);            // Acesso direto ao atributo público

        livro.preco = 39.99; // Modificação direta do atributo público
        System.out.println("Novo Preço: " + livro.preco);
        // Saída:
        // Nome do Produto: Java para Iniciantes
        // Preço: 49.99
        // Novo Preço: 39.99
    }
}

Métodos public

Um método public pode ser invocado por qualquer outro código que tenha acesso ao objeto. Estes são os métodos que definem o comportamento público de uma classe. 


class Calculadora {
    public double somar(double a, double b) { // Método público
        return a + b;
    }

    public double subtrair(double a, double b) { // Método público
        return a - b;
    }
}

public class Main {
    public static void main(String[] args) {
        Calculadora calc = new Calculadora();
        double resultadoSoma = calc.somar(15.0, 7.0);   // Invocação do método público
        double resultadoSubtracao = calc.subtrair(20.0, 5.0); // Invocação do método público

        System.out.println("Soma: " + resultadoSoma);       // Saída: Soma: 22.0
        System.out.println("Subtração: " + resultadoSubtracao); // Saída: Subtração: 15.0
    }
}

O modificador public é fundamental para construir APIs e componentes reutilizáveis em Java!
